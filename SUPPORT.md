# Support

This repository is maintained on a best-effort basis. It does not provide a
support SLA or replace LimX Dynamics commercial support channels.

## Supported scope

- DACH_TRON2A simulation using the dependency versions documented in
  `README.md`.
- Reproducible defects in the simulator, gripper test client, documentation,
  and repository CI.
- Simulation use only. Real-robot deployment and safety validation are outside
  this repository's support scope.

For usage questions and reproducible defects, open a GitHub issue and include
the simulator revision, operating system, Python version, SDK version,
robot-description revision, and sanitized logs.

Security vulnerabilities must not be reported in a public issue. Follow
`SECURITY.md`. Problems belonging to `limxsdk-lowlevel` or
`robot-description` should be reported to their respective maintainers.
